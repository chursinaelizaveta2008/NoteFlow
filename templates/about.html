О проекте NoteFlow

NoteFlow

Умное приложение для организации заметок

Возможности

  • Создание и редактирование заметок
  • Категории и цветовые метки
  • Поиск и фильтрация
  • Закрепление важных заметок
  • Архивация и восстановление
  • Подробная статистика
  • Экспорт данных

Технологии

Python

Backend

Flask

Web Framework

SQLite

База данных

Bootstrap 5

UI Framework

Разработка

NoteFlow был создан как учебный проект для демонстрации возможностей веб-разработки на Python с использованием Flask. Проект включает полный набор функций современного веб-приложения.
